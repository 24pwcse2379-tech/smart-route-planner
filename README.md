# 🚗 Smart Route Planner

## 📖 Overview

Smart Route Planner is a Data Structures project developed in C++ that helps users find the shortest or most efficient route between different locations using graph algorithms.

## ✨ Features

- Add locations and routes
- Calculate the shortest path
- Graph-based route representation
- Interactive console interface

## 🛠️ Technologies Used

- C++
- Data Structures
- Graph Algorithms
- Object-Oriented Programming

## 📂 Project Structure

```
main.cpp
graph.cpp
graph.h
route.cpp
route.h
```

## 🚀 How to Run

1. Compile all source files.
2. Run the executable.
3. Follow the on-screen menu.
